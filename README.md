## 💻 Projeto

Aplicação de recordação de memórias, onde o usuário poderá adicionar à uma timeline textos, fotos e vídeos de acontecimentos marcantes da sua vida, organizados por mês e ano.

## Tecnologias

### Mobile

- React Native
- Expo
- TypeScript
- NativeWind
- Expo Image Picker
- Expo Secure Store
- Expo Auth Session
- Expo Router

### Web

- React
- Next.js
- TypeScript
- TailwindCSS
- DayJS
- Lucide Icons
- JS Cookie

### Backend

- Node.js
- Fastify
- TypeScript
- Prisma
- Zod
